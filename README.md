# CIPGd-FSA · Módulos de Relatórios

Sistema para preenchimento de relatórios da CIPGd-FSA, com 13 módulos independentes.

## Como usar

1. Acesse o site pelo navegador
2. Selecione um módulo na tela inicial
3. Preencha os campos
4. Clique em "ENVIAR ESTE MÓDULO"
5. O relatório será copiado para a área de transferência
6. Cole no WhatsApp

## Instalação no iPhone

1. Abra o site no Safari
2. Toque no botão Compartilhar (⬆)
3. Selecione "Adicionar à Tela de Início"

## Tecnologias

- CSS3
